# Reviewer Robot

Telegram-бот для автоматического код-ревью заданий учеников. Бот слушает сообщения в группе, и по команде «проверь» отправляет код на ревью в OpenAI, а результат возвращает HTML-отчётом.

## Быстрый старт

1. Скопируй `.env.example` в `.env` и заполни переменные:

```bash
cp .env.example .env
```

2. Запусти бота:

```bash
make
```

Эта команда автоматически создаст виртуальное окружение, установит зависимости и запустит бота.

## Переменные окружения (.env)

| Переменная           | Обязательная | Описание                                      |
|----------------------|:------------:|-----------------------------------------------|
| `TELEGRAM_BOT_TOKEN` | да           | Токен бота от @BotFather                      |
| `OPENAI_API_KEY`     | да           | API-ключ OpenAI                               |
| `ALLOWED_USER_IDS`   | нет          | ID пользователей через запятую                |
| `GROUP_CHAT_ID`      | нет          | ID группы, в которой работает бот             |
| `OPENAI_MODEL`       | нет          | Модель OpenAI (по умолчанию `gpt-4o-mini`)    |

## Другие команды

```bash
make install   # только создать окружение и поставить зависимости
make clean     # удалить виртуальное окружение
```
